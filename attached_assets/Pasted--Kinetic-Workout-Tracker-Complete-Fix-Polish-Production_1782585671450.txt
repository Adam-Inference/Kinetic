FUNCTIONAL FIXES
────────────────────────────────────────

## 1. Exercise Weight Logging

Currently there is no proper way to record the weight used.

Implement:

• Numeric input field
• Decimal values allowed
• Validation
• Save button (or automatic saving)
• Editing existing value
• Persistent storage

When reopening the app, every exercise must remember its last recorded weight.

Example:

Bench Press
80

Close app

Reopen

Bench Press
80

────────────────────────────────────────

## 2. Weight Unit

Next to every weight input add a dropdown (Spinner / ExposedDropdownMenu).

Options:

• KG
• LB

Each exercise remembers its own unit.

Examples

Bench Press
80 KG

Lat Pulldown
120 LB

Do not make this global.

────────────────────────────────────────

## 3. Watch Button

The Watch button currently does nothing.

Fix it.

Requirements:

• Open saved YouTube URL
• Use Android Intent
• Prefer YouTube app
• Fallback to browser
• Gracefully handle invalid URLs
• Show user-friendly feedback if URL is missing

────────────────────────────────────────

## 4. Profile Isolation

This is extremely important.

Every profile must behave like an entirely separate user.

Verify that profiles NEVER share:

• exercises
• saved weights
• weight units
• notes
• YouTube links
• settings
• preferences
• progress

Switching profiles should instantly load that user's own data.

────────────────────────────────────────

## 5. Accent Themes

Currently these options do absolutely nothing.

Fix them.

Available accents:

• Pulse
• Ember
• Third Accent

Changing the accent should immediately update the application's appearance.

Persist the selected accent after restarting the app.

Examples:
